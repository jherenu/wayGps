Este paquete contiene un frontend completo y moderno para tu API REST de
móviles GPS. A continuación encontrarás toda la información necesaria para
instalarlo y usarlo.

   │ pip install django-cors-headers                      │
   └──────────────────────────────────────────────────────┘

2️⃣  REINICIAR SERVIDOR DJANGO
   ┌──────────────────────────────────────────────────────┐
   │ python manage.py runserver                           │
   └──────────────────────────────────────────────────────┘

3️⃣  ABRIR index.html EN TU NAVEGADOR
   
   ¡Listo! El frontend debería funcionar correctamente.

================================================================================
¿PROBLEMAS CON CORS?
================================================================================

Si ves este error:
"blocked by CORS policy" o "Access-Control-Allow-Origin"

SOLUCIÓN EN 3 PASOS:
1. pip install django-cors-headers
2. Reinicia servidor Django (Ctrl+C y volver a ejecutar)
3. Recarga el frontend (F5)

Ver: INSTALACION_CORS.md para más detalles

================================================================================
COMANDOS ÚTILES
================================================================================

Instalar dependencias:
  pip install -r requirements.txt

Iniciar Django:
  python manage.py runserver

Detener servidor:
  Ctrl+C

Ver consola del navegador:
  F12

Recargar página:
  F5

Servidor HTTP Python:
  python -m http.server 8080

================================================================================
PERSONALIZACIÓN
================================================================================

Para personalizar el frontend, edita config.js:

- URL de la API
- Intervalo de actualización automática
- Coordenadas por defecto del mapa
- Tiempo para considerar móvil "en línea"
- Colores y configuraciones visuales

No requiere conocimientos avanzados, solo edita los valores.
