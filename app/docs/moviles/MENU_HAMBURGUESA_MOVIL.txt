HTML - NAVBAR MÓVIL:
--------------------
```html
<!-- Solo visible en móviles (d-md-none) -->
<nav class="navbar navbar-dark d-md-none" style="...">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">
            <i class="bi bi-geo-alt-fill"></i> WayGPS
        </a>
        <button class="navbar-toggler" 
                type="button" 
                data-bs-toggle="collapse" 
                data-bs-target="#sidebarMenu">
            <span class="navbar-toggler-icon"></span>
        </button>
    </div>
</nav>

<!-- Sidebar con ID para collapse -->
<nav id="sidebarMenu" class="col-md-3 col-lg-2 d-md-block sidebar collapse">
    <!-- Contenido del menú -->
</nav>
```

JAVASCRIPT - AUTO-CERRAR MENÚ:
------------------------------
```javascript
function showSection(section) {
    // ... código existente ...
    
    // Cerrar menú en móviles después de seleccionar
    const sidebarMenu = document.getElementById('sidebarMenu');
    if (sidebarMenu && window.innerWidth < 768) {
        const bsCollapse = bootstrap.Collapse.getInstance(sidebarMenu);
        if (bsCollapse) {
            bsCollapse.hide();
        }
    }
}
```

CSS - ESTILOS MÓVIL:
--------------------
```css
@media (max-width: 767.98px) {

        margin-top: 56px;
    }
}
```

================================================================================
RESPONSIVE BREAKPOINTS
================================================================================

Bootstrap 5 usa estos breakpoints:

  xs: < 576px   (Móviles pequeños)
  sm: ≥ 576px   (Móviles grandes)
  md: ≥ 768px   (Tablets)
  lg: ≥ 992px   (Laptops)
  xl: ≥ 1200px  (Desktops)
  xxl: ≥ 1400px (Pantallas grandes)

NUESTRO MENÚ:
  - < 768px: Muestra navbar con botón hamburguesa
  - ≥ 768px: Muestra sidebar fijo (comportamiento original)

================================================================================
CARACTERÍSTICAS BOOTSTRAP USADAS
================================================================================

1. NAVBAR COMPONENT:
   - navbar: Contenedor principal
   - navbar-dark: Tema oscuro
   - navbar-toggler: Botón hamburguesa
   - navbar-brand: Logo/marca

2. COLLAPSE COMPONENT:
   - collapse: Componente plegable
   - data-bs-toggle="collapse": Activa el toggle
   - data-bs-target="#id": Target del elemento a plegar
   - Bootstrap.Collapse API: Control programático

3. UTILITY CLASSES:
   - d-md-none: Display none en medium y superior
   - d-md-block: Display block en medium y superior
   - Responsive: Cambia comportamiento según pantalla

PERSONALIZACIÓN
================================================================================

CAMBIAR COLOR DEL NAVBAR:

En index.html, modifica el style del navbar:
```html
<nav class="navbar navbar-dark d-md-none" 
     style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);">
```

Cambia a:
```html
style="background-color: #343a40;"  <!-- Gris oscuro -->
style="background-color: #0d6efd;"  <!-- Azul Bootstrap -->
style="background: linear-gradient(to right, #ff6b6b, #ee5a6f);" <!-- Rojo -->
```

CAMBIAR BREAKPOINT:

Si quieres que el menú hamburguesa aparezca en pantallas más grandes:

En HTML, cambia d-md-none a d-lg-none:
```html
<nav class="navbar navbar-dark d-lg-none">
```

Y en sidebar, cambia d-md-block a d-lg-block:
```html
<nav id="sidebarMenu" class="... d-lg-block ...">
```

CAMBIAR POSICIÓN DEL NAVBAR:

Para navbar fixed en la parte superior:
```html
<nav class="navbar navbar-dark d-md-none fixed-top">
```

Y ajusta el CSS:
```css
.sidebar {
    top: 0; /* En lugar de 56px */
}
```

SOLUCIÓN DE PROBLEMAS
================================================================================

PROBLEMA: Botón hamburguesa no aparece
SOLUCIÓN:
  1. Limpia cache: Ctrl+Shift+R
  2. Verifica que index.html esté actualizado
  3. Verifica viewport en DevTools (<768px)
  4. Revisa consola (F12) para errores

PROBLEMA: Menú no se abre al tocar ☰
SOLUCIÓN:
  1. Verifica que Bootstrap JS esté cargado
  2. Revisa consola para errores JavaScript
  3. Verifica data-bs-target="#sidebarMenu"
  4. Verifica id="sidebarMenu" en sidebar

PROBLEMA: Menú no se cierra automáticamente
SOLUCIÓN:
  1. Verifica que moviles.js esté actualizado
  2. Revisa función showSection()
  3. Verifica window.innerWidth < 768
  4. Limpia cache del navegador

PROBLEMA: Menú se superpone al contenido
SOLUCIÓN:
  1. Verifica z-index en CSS
  2. Verifica position: fixed
  3. Verifica main-content margin-top
  4. Recarga estilos CSS

PROBLEMA: Navbar tiene espacio arriba
SOLUCIÓN:
  1. Verifica que body no tenga margin
  2. Agrega: body { margin: 0; }
  3. Verifica no hay padding extra
